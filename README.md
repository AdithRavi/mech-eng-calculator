
# ⚙️ Mechanical Engineering Calculator
A precision-focused digital tool built to automate complex engineering computations and eliminate manual calculation errors.

## 🚀 Key Features
* **Thermodynamics:** Cycle efficiency and entropy change analysis.
* **Solid Mechanics:** Stress-strain tensors and beam deflection solvers.
* **Fluid Dynamics:** Reynolds Number and Bernoulli equation applications.
* **Machine Design:** Torque, gear ratios, and safety factor determinations.

## 🛠️ Technical Highlights
* **Logic:** JavaScript-based mathematical engines with 10-decimal precision.
* **Validation:** Built-in "Sanity Checks" to prevent physically impossible inputs (e.g., negative Kelvin).
* **UI/UX:** Clean, responsive interface designed for quick field or lab use.

## 🔧 Installation
`git clone https://github.com/AdithRavi/mechanical-engineering-calculator.git`
Then, open `index.html` in your browser.
