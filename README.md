
# ⚙️ Mechanical Engineering Calculator
**Solving the math that powers machines.**

Hey there! I'm Adith, a Mechanical Engineering student at GEC Barton Hill. I built this tool because I was tired of jumping between physical tables and basic phone calculators during my Thermodynamics and Solids labs. 

This isn't just a calculator; it's a growing suite of tools designed to handle the specific, messy formulas we deal with in 2nd-year engineering.

## 🚀 What it actually does
I've focused on the areas where I personally needed the most help with precision:

* **Thermodynamics:** Quick $PV=nRT$ solving without the unit-conversion headache.
* **Mechanics of Solids:** Calculating Stress ($\sigma$) and Strain ($\epsilon$) instantly so I can focus on the analysis, not the arithmetic.
* **Fluid Dynamics:** A Reynolds Number calculator to quickly check if a flow is Laminar or Turbulent.

## 🛠️ Built With
* **HTML/CSS:** For a clean, "No-Nonsense" interface that works on my phone and laptop.
* **JavaScript:** The math engine. I’ve tuned it for 10-decimal precision because in engineering, those small digits matter.
